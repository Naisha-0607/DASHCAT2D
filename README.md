# DASHCAT2D
A thrilling fast-paced game based with time constraints, level collection requirements (an alternative to coin collection system) and also limited to three lives per level. It is inspired by the game "STICKMAN HOOK". Includes a unique level progression system with different environments, obstacles, outfits. Audio compatibility also present.

# 🐈 DASHCAT2D

**DASHCAT2D** is a 2D platformer game developed using Unity and C#. The player controls a cat through multiple levels filled with obstacles, collectibles, interactive objects, and time-based challenges.

The goal is to navigate each level, collect the required items, avoid hazards, and reach the destination before running out of lives or time.

## 🎮 Features

- Multiple playable levels

## 🕹️ How to Play

1. Download `DASHCAT2D_NAISHA.zip`.
2. Extract the ZIP file.
3. Open the extracted folder.
4. Run the DASHCAT2D `.exe` file.
5. Play through the levels and complete the objectives.

> Keep the executable and its accompanying Unity game files in the same extracted folder.

## 🎯 Objective

Navigate through each level while avoiding obstacles and interacting with the environment. Collect the required items and complete the level within the available time while managing the player's remaining lives.



## 💻 Running the Unity Project

To open the source project:

1. Clone or download this repository.
2. Open Unity Hub.
3. Select **Add project from disk**.
4. Select the cloned `DASHCAT2D` project folder.
5. Open the project using the appropriate Unity Editor version.
